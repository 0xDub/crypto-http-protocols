# Crypto Exchange HTTP Protocol Scanner

Motivation: I wanted to see which exchanges supported QUIC

## Setup
- `rustup default stable`
- `cargo run --release`
- wait 30sec
- `BOOM, results`

### Protocols monitored:
- h1 (HTTP/1.1)
- h2 (HTTP/2)
- h3 (HTTP/3 - QUIC)
